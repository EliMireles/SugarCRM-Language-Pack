Manage License Properties

Use this page to edit or update the Sugar license information. You can enter a new application download key, the license expiration date, the number of users that the license allows, and the number of offline client licenses.

  • Click Edit to update the license information.
  • Click Save to save your changes.

When you enable automatic checks for Sugar updates, the system automatically revalidates the license when you save the information and displays the date and time in the License Validation sub-panel. To manually revalidate the license, click the Re-validate button located above the License Validation panel.

Product Types Home

Use this page to view and manage product types.

To view the details of a product type, click its name in the Product Types List.

To create a new type, click Create and enter the following information in the Product Type form:

  • Product Type. Enter a name for the product type.
  • Description. Enter a brief description of the product type.
  • Order. Enter a number to specify the order in which this category will appear in the Product Type drop-down list.
To create the product type, click Save. To create another product type, click Save and Create New.
